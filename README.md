# CodeAlpha Disease Prediction Project

## Objective
Predict breast cancer diagnosis using patient medical data, comparing linear and ensemble-based machine learning models.

## Dataset
Breast Cancer Wisconsin Dataset (Scikit-learn)

## Models Used
- **Logistic Regression:** ~97.36% Accuracy (Best Performer)
- **Random Forest:** ~96.49% Accuracy

## Key Findings
Logistic Regression performed slightly better, likely due to the linear relationships between the biological features in this specific dataset.

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
